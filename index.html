Security first

All the backups are encrypted before upload so that only you can decrypt them. The files are tested before sending and the uploaded file integrity is checked.

Working in 60 seconds

One line to copy-paste in your terminal to start the interactive install process. No configuration editing needed, everything is asked and tested live.

Backup but also restore

One command to start the interactive restore process, choose your backup from the list and let DBacked stream it back to your database.
